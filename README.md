# QR Code Generator

โปรเจกต์นี้เป็นเว็บแอปพลิเคชันที่ช่วยให้ผู้ใช้สามารถสร้าง QR Code ได้ง่าย ๆ จากข้อความที่ป้อนเข้าไป และสามารถดาวน์โหลด QR Code นั้นเป็นไฟล์ภาพ `.png` ได้

## รายละเอียดโปรเจกต์

- HTML, CSS, JavaScript
- **ไลบรารีที่ใช้**: [QRCode.js](https://github.com/davidshimjs/qrcodejs) สำหรับการสร้าง QR Code

### การใช้งาน

1. ป้อนข้อความที่คุณต้องการสร้าง QR Code ลงในช่อง input
2. คลิกปุ่ม "สร้าง QR Code"
3. ระบบจะทำการแสดง QR Code ที่สร้างขึ้น และจะแสดงปุ่ม "ดาวน์โหลด QR Code" เพื่อให้คุณสามารถดาวน์โหลด QR Code เป็นไฟล์ภาพ `.png`
